Accessibility Statement

Last Updated: June 19, 2025

Accessibility Mode: This website supports text size adjustment via browser controls

Our Commitment

DjangoMart is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying relevant accessibility standards, including WCAG 2.1 Level AA requirements.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. DjangoMart is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Recent Improvements

  • Added ARIA landmarks for screen reader navigation
  • Improved color contrast ratios to exceed 4.5:1
  • Implemented focus indicators for keyboard navigation
  • Provided text alternatives for non-text content

Technical Specifications

Accessibility of DjangoMart relies on the following technologies:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are used in conformance with the accessibility standards mentioned above.
