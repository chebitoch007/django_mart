Accessibility Statement

DjangoMart is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone.

Accessibility Features

  • Keyboard navigation support
  • Alternative text for images
  • Clear and consistent layout
  • Adjustable text size (using browser controls)

Conformance Status

We aim to conform to WCAG 2.1 Level AA standards. Our website is partially conformant with these standards.
